# JLPT N2 Trainer

Приложение для тренировки задач JLPT N2. Сделано так, чтобы любой новичок мог скачать и запустить без настройки баз данных.

## Самый простой запуск
1) Установите Node.js версии 18+ (скачать можно на nodejs.org).  
2) Скачайте проект через `git clone` или обновите уже скачанную копию командой `git pull`.  
3) В папке проекта выполните:
   ```bash
   npm install
   npm run dev:all
   ```
   - `npm run dev:all` сразу создаст или обновит локальную базу `server/jlpt.db`, запустит сервер (`http://localhost:3001`) и фронтенд (`http://localhost:5173`).
   - База `jlpt.db` хранится только у вас на диске, в git она не попадает и не ломает обновления. Если её удалить, она создастся заново при запуске.

## Что делать после git pull
- Просто запустите `npm run dev:all`. Скрипт автоматически перезальёт базу актуальными вопросами из файлов `chapter*.json`.
- Если нужно обновить базу без запуска фронтенда, выполните:
  ```bash
  npm run refresh
  ```

## Регулярные обновления
- Чтобы получать новые вопросы и исправления, периодически делайте `git pull` в папке проекта.
- После `git pull` снова запустите `npm run dev:all` (или хотя бы `npm run refresh`), чтобы база подтянула свежие данные.

## Добавление/обновление вопросов
- Все данные лежат в файлах `chapter*.json` в корне проекта. Обновили файл — выполните `npm run refresh`, чтобы база подтянула новые вопросы.
- Детальные примеры формата описаны в `docs/HOW_TO_ADD_QUESTIONS.md`.

## Админка
- Откройте `http://localhost:5173/admin`. Здесь можно просматривать и редактировать вопросы, главы и тексты для чтения.

## Если что-то сломалось с базой
- Удалите файл `server/jlpt.db`.
- Запустите `npm run refresh` или сразу `npm run dev:all`. База создастся заново из JSON-файлов.
